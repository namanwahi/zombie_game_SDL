# zombie_game_SDL
C++ implementation of a zombie game using SDL2

Libraries used:
-SDL2
-SDL_Image 2.0
-SDL_TTF 2.0

C++ version:
-C++11

Tested on Ubuntu 16.04

I have included the CMakeLists.txt file to build the project.

Controls:
W,A,S,D - move
Hold Space - aim
Release Space - shoot
